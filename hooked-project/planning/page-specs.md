# Hooked Page Specs (MVP)

## 1) Landing Page
- Goal: convert visitors into signups.
- Sections: value proposition, how it works, safety promise, creator teaser, CTA.
- Primary CTA: Create Account.

## 2) Sign Up and Verification
- Collect account details and policy consent.
- Verification status visible in account progress.
- Block adult/premium features until verified.

## 3) Onboarding
- Profile basics, interests, and discovery preferences.
- Completion checklist with progress indicator.

## 4) Discover (Swipe)
- Card-based profile discovery.
- Actions: like/dislike.
- Swipe limits enforced by account tier.

## 5) Matches
- List of mutual matches.
- Quick actions: open chat, block, report.

## 6) Chat
- Free text messaging for matched users.
- Paid media actions: image/video send or unlock.
- Clear composer states and balance checks.

## 7) Creator Profile and Content
- Subscription plans and PPV gallery.
- Tip button and premium message option.
- Explicit pricing labels before purchase.

## 8) Wallet
- Token balance, bundle purchase, transaction history.
- Downloadable receipt list.

## 9) Private Video Call
- Pre-call token check and hold.
- In-call timer, spend meter, low-balance warnings.
- Auto-end on insufficient balance.

## 10) Safety Center
- Report submission with category and evidence.
- Case status visibility for reporter.
- Abuse prevention guidance.

## 11) Settings
- Privacy controls, region blocking, pseudonym options.
- Consent and policy history.
- Billing and payment methods.

## 12) Admin Console
- Moderation queue and action log.
- Verification review status.
- Fraud flags and payout holds.
